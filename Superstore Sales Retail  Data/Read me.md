# Exploratory Data Analysis of Retail Superstore Sales

## Project Overview
This project focuses on exploratory data analysis (EDA) of retail superstore sales data to understand sales trends, customer behavior, product performance, and regional contributions. The analysis uses visual analytics to derive actionable business insights from real-world transactional data.

## Dataset Description
- Total Records: 9,800
- Total Columns: 18
- Key Features:
  - Order Date, Ship Date
  - Product Category and Sub-Category
  - Customer Segment
  - Region and State
  - Sales Amount

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Feature extraction from date columns
- Exploratory data analysis using visualizations
- Sales trend analysis across multiple years
- Category and sub-category performance analysis
- Region and state-level sales analysis
- Customer segment-based sales analysis

## Key Insights
- Sales exhibit clear seasonality with higher performance in later months of the year
- Technology is the highest revenue-generating product category across all customer segments
- Consumer segment contributes the largest share of total sales
- West region leads overall sales, driven primarily by California
- Sales are concentrated in a few high-performing sub-categories such as Phones and Chairs

## Conclusion
This project demonstrates how exploratory data analysis and visualization can be used to uncover meaningful business insights from retail sales data. The findings can support decision-making in areas such as marketing strategy, inventory planning, and regional focus.
