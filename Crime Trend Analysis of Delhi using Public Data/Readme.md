### Project Title
Crime Trend Analysis of Delhi using Public Data

### Objective
The objective of this project is to analyze crime trends in Delhi using publicly available crime records, identify long-term patterns, and demonstrate how data analysis can support evidence-based decision-making.

### Dataset
Source: Public crime records (Delhi)
Type: Year-wise crime statistics by crime category
Format: CSV

### Tools & Technologies
Python
Jupyter Notebook
Pandas (data cleaning & analysis)
Matplotlib (visualization)

### Methodology
Loaded and inspected raw crime data.
Cleaned the dataset and standardized year values.
Reshaped data from wide format to long format for analysis.
Performed exploratory data analysis using grouping and aggregation.
Created visualizations including line charts, bar charts, and a heatmap.
Interpreted trends to derive meaningful insights.

### Outcome
The project demonstrates how structured data analysis and visualization can help understand crime patterns over time and assist administrators in monitoring trends and planning interventions.
